# 🚀 NovaFin AI - Your Smart Credit Card Advisor

**NovaFin AI** is a cutting-edge, AI-powered web application that revolutionizes credit card selection for Indian users. Built with React, TypeScript, HTML, Tailwind CSS and Google's Gemini AI, it analyzes your financial profile to deliver personalized recommendations from top RBI-regulated banks like HDFC, SBI, ICICI, and Axis.

[🚀 Live Demo - NovaFin AI](https://mere-azure-jepbofbuhs.edgeone.app/) ✨
## 🌟 Key Features

- **🤖 AI-Powered Recommendation Engine**  
  Input your income, spending habits, and goals—get 4 tailored credit cards with match scores, reasoning, and direct apply links. Features 3D forms, loading animations, and comparison tables.

- **💬 Nova Chatbot & AI Assistant**  
  Your personal Nova AI companion! Chat about features, get navigation help, or dive into credit card insights. Powered by Gemini with strict NovaFin context awareness.

- **🎤 Live Voice Integration**  
  Speak your queries with real-time speech-to-text and AI responses for hands-free advice.

- **📊 Smart Comparison Tables**  
  Side-by-side breakdowns of APRs, lounge access, rewards, fees, and more across recommended cards.

- **🛡️ Privacy-First Design**  
  Local analysis where possible, no PII storage, bank-grade encryption, and one-click profile deletion.

## 🚀 Quick Start

1. **Clone & Install**  
   ```bash
   git clone <your-repo-url>
   cd novafin-ai
   npm install
   ```

2. **Add AI Key** (Optional - fallback works!)  
   Create `.env` with `VITE_GEMINI_API_KEY=your-key` for full AI power.

3. **Run Locally**  
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173)

4. **Build & Deploy**  
   ```bash
   npm run build
   npm run preview
   ```
