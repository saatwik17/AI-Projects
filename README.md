## 🌐 **How to Deploy My Projects** 

### **1. 🚀 Quick Local Setup**
```bash
git clone <your-repo-url>
cd project-name
npm install
npm run dev
```
**Open:** `http://localhost:5173` ⚡

### **2. 🚀 Production Deployment Options**

| Platform | Command | Why Use It |
|----------|---------|------------|
| **EdgeOne** | `npm run build && deploy` | **Lightning fast CDN** - My choice! |
| **Vercel** | `vercel --prod` | Git integration, auto-deploys |
| **Netlify** | Drag `dist/` folder | Free tier, instant preview |
| **GitHub Pages** | `npm run build && gh-pages -d dist` | Free, simple |

### **3. 🔑 Environment Setup** (For AI Features)
```bash
# Create .env file
VITE_GEMINI_API_KEY=your-gemini-key
VITE_OPENAI_API_KEY=your-openai-key
```
